# perfomance

Приложение report.py использует параметры --files --report.
Может быть указано несколько файлов в формате csv, как минимум один файл.
Файлы обрабатываются поочередно. Если расширение не соответствует или файл пустой или его
не возможно обработать - приложение выдаст соответствующую ошибку.

Параметр report передает наименование отчета, если его не указать - наименование будет использовано
по умолчанию.

В папке с приложением находятся примеры csv файлов, в папке screenshots - скрин запуска приложения и результат работы pytest.
Также в этой же папке находится test_report.py, для запуска тестов.
